# Novels App
Novels App is a web application that allows you to save and manage your favorite collection of novels. With this app, you can log in, view the links to all your novels, add new novels, update existing ones, delete novels, and even rate and like novels with your favorite stories. Please note that the app is currently in the beta stage.

## Accessing the App
To access the Novels App, you can visit https://novels-app.onrender.com.

## Installation
If you prefer to run the app locally, follow the steps below:

1. Clone the Git repository to your local machine using the following command:`git clone https://github.com/kipgeoffry/novels-app.git`
2. Set up the database:
    - Make sure you have MongoDB installed and running on your machine.
    - Configure the database connection in the app's configuration file.
3. Install dependencies by navigating to the project directory and running the following command:`npm install`
4. Start the application using the following command:`npm start`
5. Open your browser and visit `http://localhost:4510` to access the Novels App.

## Functionality
The Novels App provides the following functionality:

1. Login: Users can log in to their accounts to access their collection of novels.default username is admin and password is password.123.
2. View Novels: Once logged in, users can view the links to all their saved novels.
3. Add Novel: Users can add new novels to their collection by providing the necessary information.
4. Update Novel: Users can update existing novels, modifying details like the title, author, or link.
5. Delete Novel: Users can delete novels from their collection, removing them permanently.
6. Rate and Like Novels: Users can rate and like novels, expressing their appreciation for their favorite stories.
Please note that as the app is currently in the beta stage, some features might be limited or undergoing refinement
